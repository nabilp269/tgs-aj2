Food Delivery Website

Website Pemesanan Makanan Online yang Modern

Deskripsi Proyek

Food Delivery Website adalah platform pemesanan makanan online yang memudahkan pengguna mencari restoran, memesan menu favorit, dan melakukan pembayaran dengan cepat dan aman. Website ini didesain responsif sehingga nyaman digunakan di laptop maupun smartphone.

Fitur Utama:

  • Halaman beranda dengan banner promo
  • Menu makanan dengan kategori
  • Fitur keranjang belanja
  • Metode pembayaran online
  • Tracking status pesanan

Proses Design:

Dimulai dari riset kebutuhan pengguna, kemudian dibuat wireframe dan prototype menggunakan Figma. Dilakukan usability testing pada beberapa user untuk memastikan navigasi mudah dipahami sebelum pengembangan frontend.

Klien

Personal Project

Durasi

1 Bulan

Role

UI/UX Designer & Frontend Developer

Teknologi
HTML CSS JavaScript Figma
Hasil

Meningkatkan efisiensi pemesanan hingga 35% dan mendapat feedback positif dari user testing.
